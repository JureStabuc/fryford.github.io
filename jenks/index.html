Let's choose some breaks...
A common task we undertake is choosing the best way to bin our data ie group it in a sensible way. A commonly used starting point is the Jenks/Natural breaks algorithm which seeks to seeks to reduce the variance within classes and maximize the variance between classes. It's a technique used relatively frequently when mapping data.

This page allows you to load a csv file, that has each value listed in seperate rows, in a single column. The column needs a header (no gaps). You can have multiple columns and it'll return a set of bins for each column. You can tweak the number of breaks you'd like for your application.
